# Known Flaky Tests

Tests listed here are known to be intermittently unreliable. They are NOT removed from
the suite — they still run and are tracked in the audit trail. However, a single failure
of a listed test does not block a sprint release gate.

**Review cadence:** triage this file at each sprint boundary.
Remove an entry when the underlying root cause is fixed and the test has been stable for
5+ consecutive full-suite runs.
